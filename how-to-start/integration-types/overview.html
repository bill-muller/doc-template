Типи інтеграцій

Ми пропонуємо кілька способів інтеграції з нашою платіжною платформою — від повністю автоматизованих технічних рішень до простих у використанні інструментів без коду. Кожен тип інтеграції має свої переваги і підходить для різних сценаріїв бізнесу. Нижче наведено короткий опис кожного типу, а також посилання на детальні інструкції з підключення.

  • Host to Host
    Цей тип інтеграції передбачає безпосередню взаємодію між вашою серверною частиною та API нашої платформи. Підходить для компаній, які потребують гнучкості, повного контролю над платіжним процесом та мають власні технічні ресурси для розробки. Інтеграція забезпечує високий рівень автоматизації, кастомізації та захисту даних.

  • Hosted Payment Page
    Це рішення дозволяє швидко розпочати прийом платежів без необхідності самостійно розробляти інтерфейс оплати. Клієнт перенаправляється на захищену платіжну сторінку, розміщену на нашому боці, де вводить платіжні дані. Ви можете налаштувати вигляд сторінки під ваш бренд, а також отримувати повідомлення про статуси оплат.

  • Dashboard
    Панель керування платежами — це веб-інтерфейс, що дозволяє створювати платіжні запити, переглядати історію транзакцій, керувати налаштуваннями акаунта та аналітикою. Цей варіант не потребує програмування і може використовуватись навіть без інтеграції на сайт. Ідеальний для малого бізнесу, сервісів із невеликою кількістю транзакцій або для внутрішніх потреб компанії.
